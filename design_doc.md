# Objective
To provide accessible advanced hockey statistics.

# Infrastructure

Data Sources:
- NHL PBP API
- NHL Shift Charts API
- NHL Player/Team Information API

Use Python scripts to automatically query the APIs and store data in a SQLite DB.

Use Flask for back-end, Angular for front-end.

Thanks to Neil Pierre-Louis and his article https://medium.com/@neilpierre24/a-guide-to-building-your-own-full-stack-sports-analytics-website-a9247cb3b99f for the inspiration.
